Attribute - id

Attribute "id" specifies a unique id for an element. We can not have more than one element with same id in HTML

How to use id

First step: define an id with "#" character
#test {
  color: white;
  background-color: black
}

Then use "id" attribute in some element like <p> and <h1>

Hello!

<p id="test">Hello!</p>

id bookmark

we can use id as a bookmark by creating a link with a character "#" in href attribute.
Click me to teleport to id="test"
>a href="#test"<Click me to teleport to id="test"</a>

Different between class and id

A class name can be used by multiple element, but an id only can be used by one element
