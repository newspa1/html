Attribute - class

Attribute "class" is very very important. It can point to a class name in a style sheet.

How to use class

Like attribute style, class can write in the <style> element, or can write in external style sheet.

First step: define a class with "." character
.test {
  color: white;
  background-color: black
}

Then use "class" attribute in some element like <p> and <h1>

Hello!

<p class="test">Hello!</p>

If we want to use multiple classes, we can use classes with space seperate

Hello!

<p class="test test2">Hello!</p>

Different between class and id

A class name can be used by multiple element, but an id only can be used by one element
