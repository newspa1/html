CSS

CSS can be added to HTML in 3 different ways:

  • Inline - using style attribute
  • Internal - using <style> in <head>
  • External - using <link> to link to an external CSS file

    • Different way to use link

    Change the color of links

    Link has 4 different states:

    • a:link - unvisited link
    • a:visited - visited link
    • a:hover - when mouse point to the link
    • a:active - active link

    Change link to a button

    We can have the below button by using code:

    a:link, a:visited {
    background-color: red;
    color: white;
    padding: 15px 25px;
    text-align: center;
    text-decoration: none;
    display: inline-block;
    }
    a:hover, a:active {
    background-color: pink;
    }
