Element - ul, ol and dl

<ul> - unordered list

Each list item starts with <li> element

CSS "list-style-type" property defines the style of the marker, there have 4 types:

  • disc - set marker to a bullet(default)
  • circle - set marker to a circle
  • square - set marker to a square
  • none - the list will not have marker

<ol> - ordered list

Each list item starts with <li> element

Attribute "type" of <ol> element, defines the type of the marker

  • type="1" - numbered with number(default)
  • type="A" - numbered with uppercase letter
  • type="a" - numbered with lowercase letter
  • type="I" - numbered with uppercase roman letter
  • type="i" - numbered with lowercase roman letter

<ol> counts from 1 by default. If we want to count from specified number, we can use attribute "start" of <ol>

<dt> - description list

Each list item contains two elements <dt> and <dd>, represent the term(name) and describes. For example:

A
- The first letter in English.
B
- The second letter in English.
C
- The third letter in English.
