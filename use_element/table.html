Element - table, tr, th and td

Basic form of table:

One Two Three
1 2 3
<table>
  <tr>
    <th>One</th>
    <th>Two</th>
    <th>Three</th>
  </tr>
  <tr>
    <td>1</td>
    <td>2</td>
    <td>3</td>
  </tr>
</table>

Table with CSS

border - add a border
  border: 1 solid black;
border-collapse - avoid having double borders
  border-collapse: collapse
border-radius - the borders get round corners:
  border-radius: 10px
border-style - set the appearance of the border
  border-style: dotted
border-color - set the color of the border
  border-color: red
border-spacing - set the space between table cells
  border-spacing: 15px padding - add padding on table cells
  padding: 15px

Make zebra stripes

To make the zebra stripes in the table, we can use below 3 element:

  • tr:nth-child(even/odd) - horizontal stripes
  • th:nth-child(even/odd) and td:nth-child(even/odd) - vertical stripes

Use "colgroup" element in the table

We can use <colgroup> to define multiple column in the same style by using <col> element inside.

SUN MON TUE Wed THU FRI SAT
13 14 15 16 17 18 19

Only "width", "visibility", "background" and "border" can be used in <col>
